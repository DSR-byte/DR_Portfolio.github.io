Automated Texting Service

I designed and built a macOS application that enables mass texting via a connected phone.
During this project, I learned to work with various Python libraries for reading and writing Excel files. The application allows users to upload a file containing names, phone numbers, and addresses.
It then sends personalized text messages using the corresponding names and, optionally, addresses.
The tool is versatile and can be applied in multiple contexts; it is currently being used by a client for real estate marketing and by a local college club for announcements.
