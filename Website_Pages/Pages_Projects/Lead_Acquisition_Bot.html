Lead Acquisition Bot

I designed and built a macOS application that enables mass lead acquisition.
During this project, I worked with various Python libraries that enable browser emulation. The application allows users to upload a file containing names and addresses.
It then scrapes publicly available data to find a phone number for each name.
The tool is versatile and can be applied in multiple contexts; it is currently being used by a client for real estate marketing.
