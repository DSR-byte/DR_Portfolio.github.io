Built a Functioning CPU from Scratch in SystemVerilog

I designed and built a custom CPU from scratch using SystemVerilog.
The project involved implementing all major components of a functioning processor, including an ALU, register file, immediate generator, branch logic, and control unit.
I designed the architecture to support instruction decoding, pipelined execution, and basic branching, and simulated each module to verify correctness.
This hands-on experience gave me a strong understanding of digital logic design, data path control, and instruction flow, as well as how to structure modular, synthesizable HDL code.
Through this work, I developed a solid grasp of the full ASIC design flow at the RTL level.
